Library: Animation.js
Overview
anim package Animation, Manager Released under Academic Free Licence 3.0.
  • source: Animation.js
  • author: Garrett Smith

    Animation is purely a timing framework that passes a position to the run() method. It has nothing to do with HTML elements, CSS, et c.

    anim.Manager manges running animations in two threads. This can be changed by setting APE.anim.Manager.isDoubleThreaded = false;

    anim.Transitions contains effects for speed/timing, such as acceleration and easing. 

     
b = new APE.anim.Animation( "blah" );
b.run = function(position) { // <-- you implement run.

}
b.start() // <-- then call start.

    anim.Transition

    All functions work with a position (0-1).
