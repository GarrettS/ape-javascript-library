APE.dom.StyleSheetAdapter.getRule(selectorText)
IE will not return the correct selector text for Element selectors in HTML. Instead, IE puts the element selector and pseudoclass selectors in upper case. This function compares selectors, case sensitive, but case insensitive for element selector parts. Does not traverse into media blocks.
