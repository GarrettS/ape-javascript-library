New!

APE development is now hosted on Github and has lighthouse bug tracker.

APE does not change the way you write JavaScript. APE uses a global namespace, APE, and minimal inheritance.

APE is a good library for building a JavaScript library on top of.

Features

  • APE Core (< 2k minified) Namespacing and Object creational features.
  • EventPublisher - AOP custom and native events.
  • Subscription Based Async Requests - use APE.EventPublisher to chain multiple callbacks.
  • Time-based animation.
  • Featureful drag and drop.

Getting Started

The commonly used files, APE.js, EventPublisher.js, and dom.js, have been conveniently packaged in to one rollup file, ape-ep-dom.js (< 10k minified).

For css-based animation, the files StyleTransition.js and Animation.js have been rolled up into anim.js (< 7k minified).
